Post'Em: Resumo

O que faz

Usando a ferramenta Post'Em, você pode fazer upload delimitado de planilha (CSV) para apresentar de feedback para os participantes do site.

Post'Em fornece as seguintes funcionalidades:

  • Exibe feedback de um participante individual ao participante

  • Exibe comentários de todos os participantes para o líder do Site

  • Permite que o líder do site visualize um único feedback do participante

  • Calcula e exibe estatísticas simples para os dados numéricos

  • Atualize uma arquivo de feedback existente

  • Upload de mais de um arquivo de feedback

Principais conceitos

Formatação de Planilha: Para que um arquivo de planilha seja compatível com Post'Em, ele deve ser salvo no formato .csv. A primeira coluna deve conter usernames, e a primeira linha devem conter títulos. Outras células de dados podem conter texto ou valores numéricos.

Considerações

  • Um líder do site de projeto pode usar Post'Em para dar feedback aos membros sobre as suas contribuições para um relatório, feedback compartilhado de outros membros, e fornecer e acompanhar metas específicas para cada indivíduo.

  • Um líder do site de curso pode usar Post'Em para dar feedback aos alunos sobre o seu desempenho em sala de aula, que pode incluir notas, porcentagens, notas de letras e/ou comentários.
